# Moon Weight Log

Moon Weight Log 是一个本地优先的体重、围度、饮食与运动趋势记录 PWA。它面向个人单机使用，不需要账号，不登录，不联网，不上传云端，也不提供共享、排行、PK 或社交功能。

## 隐私与数据

- 数据默认保存在当前设备当前浏览器的 `localStorage`。
- 换手机、清理浏览器数据、卸载浏览器、隐私模式或系统清理都可能导致数据丢失。
- 请定期在「数据管理」里导出 JSON 备份。
- JSON 备份包含目标设置、体重记录、围度记录和应用设置。

## 使用方式

直接打开 `index.html` 即可使用。为了获得完整 PWA 离线缓存体验，建议在本地用一个静态文件服务器打开，例如：

```bash
python3 -m http.server 8000
```

然后访问：

```text
http://localhost:8000
```

## 添加到手机主屏幕

- iPhone Safari：打开页面，点分享按钮，选择「添加到主屏幕」。
- Android Chrome：打开页面，点菜单，选择「安装应用」或「添加到主屏幕」。

首次访问后，应用会缓存 `index.html`、`styles.css`、`app.js`、`manifest.json` 和图标文件，用于基础离线访问。

## 导入 JSON 备份

进入「数据管理」，选择「导入 JSON 备份」。导入会用备份内容替换当前浏览器中的本机数据，导入前建议先导出当前 JSON。

## 导出 CSV

进入「数据管理」，可以分别导出：

- 体重记录 CSV
- 围度记录 CSV

CSV 可用于表格软件中查看或自行分析。

## 重要说明

Moon Weight Log 不是医疗诊断工具，不替代医生、营养师或治疗师建议。它只帮助记录个人趋势与阶段观察，不用于 BMI 评价，不自动判断身体状态，也不建议用于极端节食或医疗决策。如果你记录到持续异常或明显不适，请联系专业人士判断。
